# 文献中模型模拟方式总结

## 一、基础模型类别

### 1. 自旋/XY模型（Continuous Spin Models）

#### 1.1 经典XY模型（平衡态）
**代表文献**: Mermin & Wagner (1966), Hohenberg (1967)

**模型定义**:
- **维度**: d = 1, 2, 3...
- **自由度**: 平面转子/XY自旋 Sᵢ ∈ ℝ²，|S| = 常数，取向角 θᵢ ∈ [0, 2π)
- **相互作用**: 短程铁磁耦合 J Σ⟨ij⟩ cos(θⱼ - θᵢ)
- **动力学**: 平衡统计力学（配分函数 Z = ∫ Dθ exp(-βH)）
- **对称性**: O(2) / U(1) 连续旋转对称性

**模拟方法**:
- Monte Carlo (Metropolis, Heat-bath)
- Langevin动力学: γ∂ₜθᵢ = -δH/δθᵢ + ξᵢ(t)

**关键结论**: d ≤ 2 时不存在真正长程序（MW定理）

---

#### 1.2 非互易XY模型
**代表文献**: Loos et al. (2023), Lorenzana et al. (2025)

**模型定义**:
- **视野锥机制**: 每个自旋只与位于其当前朝向"视野锥"θ内的近邻耦合
- **相互作用矩阵**: Jᵢⱼ(φᵢ) ≠ Jⱼᵢ(φⱼ) —— 非对称
- **动力学**: Monte Carlo Glauber动力学（非平衡）

**模拟方法**:
- 构型依赖的键选择（configuration-dependent bond selection）
- 非平衡Monte Carlo更新

**打破MW机制**: 
- 构型依赖的键稀释 → 有效U(1)对称性被打破
- 长波自旋波能量在热力学极限下发散（E_tot ~ L²ηJ → ∞）

---

#### 1.3 自驱动转子模型
**代表文献**: Rouzaire & Levis (2021)

**模型定义**:
- **自由度**: 角度 θᵢ
- **非平衡机制**: 随机本征频率 Ωᵢ ~ N(0, σ²) —— 每个自旋独立注入能量
- **动力学方程**: 
  ```
  θ̇ᵢ = Ωᵢ + J Σⱼ sin(θⱼ - θᵢ) + √(2D)νᵢ(t)
  ```

**模拟方法**:
- 数值积分（Runge-Kutta等）
- 匈牙利算法匹配缺陷对

**结果**: 自旋转破坏QLRO，形成有限大小畴，缺陷超扩散

---

### 2. 活性物质模型（Active Matter）

#### 2.1 Vicsek模型
**代表文献**: Vicsek et al. (1995)

**模型定义**:
- **维度**: 2D
- **自由度**: 粒子位置 rᵢ，速度方向 θᵢ，速度大小 |v| = v₀ = 常数
- **相互作用**: 短程对齐 — 取半径r内邻居的平均运动方向
- **动力学**: **非Hamiltonian确定性更新规则**
  ```
  θᵢ(t+1) = ⟨θ(t)⟩ᵣ + Δηᵢ  (η ∈ [-π/2, π/2] 为随机扰动)
  rᵢ(t+Δt) = rᵢ(t) + vΔt · ê(θᵢ)
  ```

**模拟方法**:
- **离粒子Euler积分**（分子动力学风格）
- 周期性边界条件
- 系统尺寸 N = 300-3000

**关键特征**:
- 非平衡（违反细致平衡）
- 运动学相变: η_c ∝ ρ^{-α}
- 序参量 ⟨v⟩ ∝ (η_c - η)^β，β ≈ 0.45（非标准指数）

---

#### 2.2 Toner-Tu流体动力学方程
**代表文献**: Toner & Tu (1995)

**模型定义**:
- **维度**: 一般d维（重点d=2）
- **自由度**: 连续速度场 v(r,t)，|v|² = σ₀/ρ₀
- **动力学方程** (非平衡Langevin方程):
  ```
  ∂ₜv + (v·∇)v = αv - β|v|²v - ∇P + D₁∇²v + D₂∇(∇·v) + λ(v·∇)v + f
  ∂ₜρ + ∇·(ρv) = 0
  ```

**理论方法**:
- **场论 + 重整化群 + 标度分析**
- 关键创新: 对流项 (v·∇)v 使动力学非势，等效于长程作用
- 临界维度 d_c = 4（> 平衡XY模型的 d_c = 2）

---

#### 2.3 干燥对齐稀疏活性物质（DADAM）
**代表文献**: Chaté (2020)

**模型定义**:
- **自由度**: 粒子位置 rᵢ，取向 θᵢ
- **连续时间动力学**:
  ```
  ṙᵢ = ±v₀ ê(θᵢ)
  θ̇ᵢ = (K/nᵢ) Σⱼ sin[k(θⱼ - θᵢ)] + ξᵢ
  ```
- **三类模型**: 
  - 极性/Vicsek类 (k=1, a=0)
  - 自推进棒 (k=2, a=0)  
  - 活性向列类 (k=2, a≠0)

**理论方法**:
- 微观模型 + Toner-Tu水动力学理论
- Boltzmann-Ginzburg-Landau方法推导

---

### 3. 剪切流/层流驱动模型

#### 3.1 剪切流下的O(2)模型
**代表文献**: Nakano, Minami & Sasa (2021)

**模型定义**:
- **维度**: 2D ([0, Lₓ] × [0, Lᵧ])
- **自由度**: 标量O(2)场 φ(r,t) = (φ₁, φ₂)
- **相互作用**: 短程φ⁴ Landau自由能
  ```
  Φ[φ] = ∫ d²r [(κ/2)(∇φₐ)² + (r/2)|φ|² + (u/4)|φ|⁴]
  ```
- **非平衡机制**: 稳态均匀剪切流 v(r) = (γ̇y, 0)
- **动力学方程**: **时间依赖Ginzburg-Landau模型（Model A + 对流）**
  ```
  ∂ₜφₐ + v·∇φₐ = -Γ δΦ[φ]/δφₐ + ηₐ
  ```
  噪声关联: ⟨ηₐ(t,r)η_b(t',r')⟩ = 2ΓT δₐ_b δ(t-t') δ(r-r')

**模拟方法**:
1. **线性化相涨落分析** → 解析得到关联函数 C_θθ(k) ~ |k|^{-2/3}
2. **有限尺寸标度分析**（Binder累积量交叉）→ 确认二级相变
3. **直接数值求解完整非线性Langevin方程** (有限差分/谱方法)

**关键结果**: 
- 相位涨落从 |k|^{-2} 被抑制为 |k|^{-2/3}（沿流向方向）
- 各向异性标度指数: zₓ = 3, χ = -d/2

---

#### 3.2 一般层流驱动模型
**代表文献**: Minami & Nakano (2022)

**模型定义**:
- **非平衡机制**: 线性层流 v(x) = D·x，其中
  ```
  D = [[0, S+A], [S-A, 0]]
  ```
  - 旋转流 (S < A): 纯旋转
  - 剪切流 (S = A): 经典剪切
  - 拉伸流 (S > A): 存在膨胀/压缩

**理论方法**:
- **1/N展开**（大N极限）精确求解
- **相互作用表象**（interaction picture）分析波矢沿流线的动力学
- 积分NG模式关联判断是否红外发散

**关键发现**:
- 只有能产生超扩散的流动（剪切+拉伸）才能稳定长程序
- 纯旋转流不能稳定长程序

---

#### 3.3 标度分析统一框架
**代表文献**: Ikeda (2024)

**理论方法**:
- **简单标度分析**: 尺度变换
  ```
  x₁ → b^{ζ₁}x₁, t → b^z t, φ → b^χφ
  ```
- 线性平流（剪切流）: z = 2, ζ₁ = 3（沿流）, ζ₂ = 1（垂直流）, χ = -d/2
- 上临界维度 d_up = 2 → d=2 即获得平均场临界指数

---

### 4. 弹性/晶体非平衡驱动模型

#### 4.1 配对活性应力模型
**代表文献**: Keta & Henkes (2024)

**模型定义**:
- **维度**: 2D
- **自由度**: 矢量位移场 u(r,t)
- **相互作用**: 弹性相互作用（bulk modulus B + shear modulus μ）
- **非平衡机制**: **配对随机活性应力**（pair-wise fluctuating active stresses）
  - 力以等大反向作用于粒子对
  - 守恒质心
- **动力学方程**:
  ```
  ζ u̇(r,t) = -∫ d²r' D^el(r-r')u(r',t) + ∇·σ^act(r,t)
  ```
  活性应力噪声关联: ⟨λ(r,t)·λ(r',t')⟩ = -σ²a² e^{-|t-t'|/τ} ∇²δ(r-r')

**模拟方法**:
- **线性弹性理论**（Fourier空间解析计算）
- **数值模拟**: junctional tension vertex model (jtVM) 和 pair ABP model (pABP)

**打破MW机制**: 
- 质心守恒 → 噪声谱 ∝ |q|²（长波长处趋于零）
- 抑制长波位移涨落 → 位移方差有限

---

#### 4.2 双浴驱动晶体模型
**代表文献**: Maire & Plati (2024)

**模型定义**:
- **维度**: 2D周期晶格 L×L
- **自由度**: 粒子位移矢量 uₙ（每个格点2D位移）
- **相互作用**: 简谐最近邻耦合
- **非平衡机制**: **双浴耦合**
  1. 全局浴（不守恒动量）: F_com = -γ_com u̇ₙ + √(2γ_com T_com)ξₙ — 外部能量注入
  2. 局部浴（守恒动量和质心）: F_loc — 类Model B

**模拟方法**:
- **简谐晶体解析模型** → Fourier空间中位移关联精确求解
- **连续时间随机组织模型**（硬盘 + 活性碰撞 + 热浴）→ 数值模拟

**关键洞察**: 平移长程序稳定性由**质心温度 T_com** 控制

---

### 5. 振子/相位模型

#### 5.1 局域耦合Kuramoto模型
**代表文献**: Sakaguchi et al. (1987) (SSK)

**模型定义**:
- **维度**: d = 1, 2, 3 超立方晶格
- **自由度**: 相位 φᵢ (mod 2π)
- **相互作用**: 最近邻正弦耦合
- **动力学方程**:
  ```
  φ̇ᵢ = ωᵢ + K Σ_{<ij>} sin(φⱼ - φᵢ)
  ```
- **无序**: 固有频率 ωᵢ 服从正态分布 g(ω)（淬火无序）

**模拟方法**:
- Euler方法数值积分
- Fourier分析相位差的模式

**关键结果**:
- d ≤ 2: 完美同步不可能，∫ dq·q^{d-3} 在 d≤2 时发散
- d = 3: 可能存在宏观同步

---

#### 5.2 频率分布依赖的振子模型
**代表文献**: Daido (1988)

**模型定义**:
- **维度**: 一般 d 维空间晶格
- **相互作用**: 有限范围耦合 h_{ij}(θⱼ - θᵢ, ε)
- **无序**: 固有频率分布 f(Ω)，尾部行为 f(Ω) ~ |Ω|^{-1-α} (|Ω| → ∞)

**理论方法**:
- **实空间重整化群（RG）**: 定义"块振子"（block oscillator）
- **稳定分布理论**: 块振子有效频率服从α-稳定分布

**关键结果**:
- 下临界维度: d_L ≥ α/(α-1)（当 1 < α ≤ 2）
- α = 1（Lorentzian分布）: **任何有限维度无宏观同步**
- α = 2（高斯分布）: d_L ≥ 2

## 二、模拟方法论分类总结

### 1. 数值模拟方法

| 方法类型 | 适用模型 | 代表文献 | 特点 |
|---------|---------|---------|------|
| **Monte Carlo** | XY模型、Ising模型 | Loos et al. | Glauber/Metropolis更新，平衡/非平衡 |
| **分子动力学** | 粒子系统（Vicsek类） | Vicsek et al., Chaté | 离粒子Euler积分，确定性+随机 |
| **Langevin动力学** | 连续场论 | Nakano 2021, Toner-Tu | 有限差分/谱方法求解SPDE |
| **顶点模型** | 生物组织、活性晶体 | Keta & Henkes | 针对细胞/粒子网络拓扑 |
| **随机组织模型** | 硬盘系统 | Maire & Plati | 事件驱动碰撞+热浴 |
| **RG数值方法** | 场论 | Codello & D'Odorico | 迭代求解标度势方程 |

### 2. 解析理论方法

| 方法类型 | 适用场景 | 代表文献 | 核心工具 |
|---------|---------|---------|---------|
| **Bogoliubov不等式** | 严格证明无长程序 | Mermin & Wagner, Hohenberg | 算符对易关系 + Cauchy-Schwarz |
| **Fourier分析** | 线性涨落分析 | SSK 1987, Nakano 2021 | q-空间积分发散判断 |
| **重整化群（RG）** | 临界行为、普适类 | Toner & Tu, Daido 1988 | 标度变换、不动点分析 |
| **功能RG（FRG）** | 连续维度分析 | Codello 2013 | Wetterich方程、有效平均作用量 |
| **标度分析** | 确定临界维度、指数 | Ikeda 2024 | 量纲分析、标度变换 |
| **相互作用表象** | 层流动力学 | Minami & Nakano 2022 | 类似量子力学的演化分解 |

### 3. 检验长程序存在的诊断工具

| 诊断量 | 定义 | 长程序判据 |
|-------|------|-----------|
| **磁化强度/序参量** | m = |⟨e^{iθ}⟩| | m → 常数 > 0（L → ∞） |
| **自旋-自旋关联** | C(r) = ⟨Sᵢ·Sⱼ⟩ | C(r) → 常数 > 0（r → ∞） |
| **Binder累积量** | U_L = 1 - ⟨m⁴⟩/(3⟨m²⟩²) | U_L 交点确定相变点 |
| **位移方差** | u² = ⟨u²⟩ | 有限（vs ~log L发散） |
| **结构因子** | S(q) = ⟨|ρ_q|²⟩ | 超均匀性: S(q) ~ q^η（η > 0） |
| **缺陷均方位移** | MSD ~ t^α | 正常扩散: α=1; 超扩散: α>1 |
| **同步序参量** | r = |⟨e^{iφ}⟩| | r → 有限值（热力学极限） |

## 三、打破MW定理的机制分类

| 机制类别 | 物理本质 | 关键特征 | 代表文献 |
|---------|---------|---------|---------|
| **剪切/层流平流** | 对流项 (v·∇)φ 打破时间反演 | k^{-2/3} 抑制, 超扩散 | Nakano 2021, Minami 2022 |
| **配对活性应力** | 质心守恒, 噪声谱 ∝ q² | 位移方差有限 | Keta & Henkes 2024 |
| **非互易相互作用** | Jᵢⱼ ≠ Jⱼᵢ, 构型依赖 | 有效对称性改变, U(1)打破 | Loos 2023, Lorenzana 2025 |
| **自驱动/平流非线性** | λ(v·∇)v 等效长程 | 各向异性标度 χ < 0 | Toner & Tu 1995, Chaté 2020 |
| **双浴选择性加热** | 质心温度独立控制 | T_com → 0 时 LRO | Maire & Plati 2024 |

## 四、维度与序的关系

| 系统类别 | 下临界维度 | d=2行为 | 上临界维度 |
|---------|-----------|--------|-----------|
| 平衡O(N≥2) | d_L = 2 | QLRO (BKT) | d_c = 4 |
| 非互易XY | - | LRO (条件性) | - |
| Vicsek/活性物质 | - | LRO | d_c = 4 |
| 剪切流O(N) | - | LRO | d_up = 2 |
| 配对活性应力 | - | LRO | - |
| 振子系统(α=2) | d_L = 2 | 可能无MME | - |
